Live Server автоматически обновляет страницу в браузере при изменении HTML-файла, поэтому не нужно каждый раз вручную перезагружать страницу, как при обычном открытии через проводник. Это ускоряет разработку и позволяет сразу видеть изменения.

Тег `<main>` используется для обозначения **основного содержимого страницы**. Внутри него размещается главная информация сайта, которая отличается от меню, шапки (`header`) и подвала (`footer`).

Команда `git clone` нужна для **копирования удалённого репозитория с GitHub на компьютер**. Она создаёт локальную папку проекта вместе со всей историей изменений.

Атрибут `id` используется для **уникальной идентификации HTML-элемента** на странице. По `id` можно обращаться к элементу в CSS (`#id`) или JavaScript (`getElementById`).

Теги заголовков `<h1>`–`<h6>` отличаются **уровнем важности и размером текста**. `<h1>` — самый главный заголовок страницы, а `<h6>` — наименее важный и самый маленький.
